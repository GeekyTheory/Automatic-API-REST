Automatic API Rest

Simplify your APPs

WHAT'S THIS?

Automatic API Rest is an Open Source Tool to provide you a full API Rest of your Data Base. You can get information externally of your Data Base to interact with your Apps. Automatic API Rest allows you to build a JSON and XML trees of your Data Base. You can choose which tables are public and which are private

Open Source Project

Automatic API Rest is an Open Source Development Project. If you want to add your owns lines code, you will be able to participate with this project. Do a Fork and enjoy!
